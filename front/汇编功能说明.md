# 汇编功能 - 首页入口配置

## 需要添加的图标

请在 `front/src/assets/img/` 目录下添加一个新的按钮图标：
- 文件名：`btn11.png`
- 建议尺寸：与其他按钮图标一致
- 建议样式：书本或文件夹图标

## 首页配置

在 `front/src/views/Home.vue` 中添加汇编按钮：

### 1. 导入图标（在文件顶部）

```javascript
import btn11 from '@/assets/img/btn11.png'  // 汇编
```

### 2. 添加到右侧按钮数组

在 `rightButtons` 数组中添加：

```javascript
const rightButtons = [
  { id: 1, path: '/upload', title: '材料上传', img: btn9, roles: ['admin', 'inspector'] },
  { id: 2, path: '/checklist', title: '报告清单', img: btn6, roles: ['admin', 'inspector', 'leader', 'top_viewer'] },
  { id: 3, path: '/report', title: '报告预览', img: btn7, roles: ['admin', 'inspector', 'leader', 'top_viewer'] },
  { id: 4, path: '/archive', title: '月度归档', img: btn10, roles: ['admin', 'inspector', 'leader', 'top_viewer'] },
  { id: 5, path: '/compilation', title: '汇编', img: btn11, roles: ['admin', 'inspector', 'leader', 'top_viewer'] }  // 新增
]
```

## 功能说明

- **所有角色都可以访问**：检察官、分管领导、院领导、管理员
- **只有管理员可以上传和管理**：上传文档、编辑文档、删除文档、管理分类
- **其他用户只能查看和下载**：浏览文档、搜索文档、下载文档、预览PDF

## 数据库初始化

运行以下 SQL 文件创建数据库表：

```bash
mysql -u root -p paizhu_db < backend/migrations/create_compilation_tables.sql
```

或者在 MySQL 客户端中执行：

```sql
source backend/migrations/create_compilation_tables.sql;
```

## 测试步骤

1. 重启后端服务
2. 刷新前端页面
3. 以管理员身份登录
4. 点击"汇编"按钮
5. 测试上传文档、管理分类等功能
