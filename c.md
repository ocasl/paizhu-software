如果全程使用**纯网页端**（平板+电脑均通过浏览器访问，无原生APP），核心技术栈需围绕「响应式适配（平板+电脑）、离线功能支持（无网填表格）、数据自动收集、报告生成、银河麒麟服务器兼容」展开，以下是一套**成熟稳定、贴合需求**的纯网页端技术栈方案，覆盖前端、服务端、数据存储、部署全流程：

### 一、核心技术栈总览（按模块划分）

### 二、各模块技术选型细节（为什么选+核心用途）
#### （一）前端技术栈（核心：适配+离线+文件处理）
1. **核心框架：Vue 3 + Vite**
   - 优势：轻量高效、编译速度快，支持响应式开发，生态成熟，适配平板/电脑双端；Vite比Webpack更适合开发调试，迭代效率高。
   - 核心用途：搭建整个网页端的基础架构，管理页面组件、路由（Vue Router）、全局状态（Pinia，如用户信息、离线状态）。

2. **UI组件库：Element Plus**
   - 优势：适配Vue 3，提供丰富的办公类组件（表格、表单、上传、弹窗、下拉选择等），支持响应式布局，可快速实现平板端触摸友好的界面（如放大按钮、适配横屏的表格布局）。
   - 核心用途：快速搭建「日/周/月检察记录表单」「材料上传入口」「报告预览界面」「线上审阅页面」，无需自定义开发复杂组件。

3. **离线能力核心：PWA + IndexedDB + Service Worker**
   - 核心逻辑：通过PWA插件（`vite-plugin-pwa`）将网页核心资源（HTML/CSS/JS、表格模板）缓存到本地；用IndexedDB存储离线填写的表格数据（如工作日志、谈话数量）；Service Worker拦截网络请求，无网时直接读取本地缓存，联网后自动同步。
   - 核心用途：满足“无网制作表格、暂存附件”需求，确保平板在监狱无网络区域可正常操作，联网后数据自动上传到银河麒麟服务器。

4. **文件处理：vue-upload-component + JSZip**
   - vue-upload-component：支持大文件分片上传（如视频、多份笔录PDF）、断点续传，适配平板网络不稳定场景，避免上传中断。
   - JSZip：实现“一键压缩归档”功能，将月度报告、日志、附件等打包为“XX检察室+年+月”压缩包，支持下载/上传到服务器。

5. **其他辅助工具**
   - Axios：处理前端与服务端的HTTP/HTTPS请求，添加请求拦截（携带JWT令牌鉴权）、响应拦截（处理离线/错误状态）。
   - vue-signature-pad：实现平板触摸手写签名（如谈话笔录签字），生成图片后随材料上传。
   - 媒体API（getUserMedia）：调用平板摄像头，支持现场拍照/拍视频，暂存本地后联网上传。

#### （二）服务端技术栈（核心：数据整合+报告生成+银河麒麟兼容）
1. **核心框架：Spring Boot 3.x**
   - 优势：Java生态成熟，跨平台兼容性强（完美适配银河麒麟Linux系统），开发效率高，支持与前端、数据库、中间件无缝对接。
   - 核心用途：提供RESTful API供前端调用（如提交表格数据、上传附件、获取报告草稿），处理核心业务逻辑（数据统计、报告生成、权限校验）。

2. **数据操作：MyBatis-Plus + 人大金仓/MySQL**
   - 人大金仓 KingbaseES V8R6（推荐）：国产数据库，官方适配银河麒麟系统，兼容MySQL语法，满足检察系统国产化/合规要求，存储表格结构化数据（如罪犯分类数量、检察次数、谈话记录统计）。
   - MyBatis-Plus：简化数据库CRUD操作，快速查询/统计前端提交的数据（如按月份统计监控检察次数、按类型统计谈话数量）。

3. **报告生成：Apache POI + Freemarker**
   - Apache POI：解析前端上传的Excel花名册（如入监/出监人员名单），提取数据；同时支持生成Word版《月度工作情况报告》，填充模板占位符。
   - Freemarker：定制报告模板（按需求中的Word格式），将服务端统计的数据（如在押罪犯数、纠正违法通知书份数）动态填充到模板，生成最终报告。

4. **中间件：Redis + RabbitMQ（银河麒麟兼容版）**
   - Redis 7.x for Kylin：缓存热点数据（如用户权限、报告草稿、罪犯分类统计结果），提升前端查询速度；存储离线同步任务队列（如联网后待上传的表格/附件）。
   - RabbitMQ for Kylin：处理异步任务（如大文件上传后的合并、报告生成），避免阻塞服务端主线程，提升系统稳定性。

5. **安全与合规：Spring Security + JWT + HTTPS**
   - Spring Security + JWT：实现用户身份认证（平板端/电脑端登录）、权限控制（一线人员仅能填表格，领导仅能审阅），Token加密传输。
   - HTTPS：通过Nginx配置SSL证书，加密前端与服务端的所有通信，符合检察系统数据安全要求。
   - 操作日志：Logback记录所有关键操作（如填写表格、上传附件、审阅报告），支持审计追溯。

#### （三）数据存储（核心：安全存储+银河麒麟适配）
1. **结构化数据：人大金仓/MySQL**
   - 存储所有表格的核心字段数据（如《派驻检察工作日志》《事项清单》的填写内容、统计后的报告字段），支持事务和主从复制，确保数据一致性和备份。

2. **非结构化数据：MinIO for Kylin**
   - 优势：开源对象存储，原生支持Linux（银河麒麟），部署灵活，支持文件加密存储、权限控制，适配检察工作的敏感附件（照片、视频、笔录PDF）存储需求。
   - 核心用途：存储前端上传的所有附件，与表格数据关联（如某条日志对应的监控异常照片），支持后续下载、归档。

3. **缓存：Redis 7.x for Kylin**
   - 缓存高频访问数据（如月度报告草稿、用户权限信息），减少数据库查询压力，提升网页端响应速度。

#### （四）部署/DevOps（核心：银河麒麟兼容+稳定运行）
1. **容器化：Docker CE for Kylin + Docker Compose**
   - 将前端、服务端、数据库、中间件等组件打包为Docker容器，统一部署环境，避免“开发环境能跑、银河麒麟服务器跑不通”的兼容性问题。
   - 通过Docker Compose编排容器，一键启动/停止所有服务，简化部署流程。

2. **Web服务器：Nginx for Kylin**
   - 反向代理服务端API，隐藏服务器真实IP，提升安全性；
   - 配置HTTPS，加密数据传输；
   - 静态资源托管（如前端打包后的HTML/CSS/JS、报告模板），提升访问速度。

3. **安全合规工具**
   - SonarQube for Kylin：代码质量检测，修复安全漏洞，确保代码符合检察系统合规要求；
   - 银河麒麟安全审计模块：开启系统自带的审计功能，与应用操作日志联动，强化可追溯性。

### 三、技术栈核心适配要点（纯网页端关键保障）
1. **平板端适配**：通过Element Plus的响应式布局+CSS Media Query，优化平板触摸体验（按钮间距≥10px、输入框高度≥44px、横屏优先显示表格）；禁用hover效果（平板无鼠标悬浮）。
2. **离线功能保障**：前端通过IndexedDB存储表格数据，用Cache API缓存核心资源，Service Worker拦截请求；服务端设计“离线任务队列”，联网后按顺序同步数据，避免冲突。
3. **银河麒麟兼容性**：所有组件（JDK、数据库、中间件、Docker）均选择“银河麒麟官方认证版本”，优先使用aarch64/x86_64对应架构的安装包，避免依赖缺失或内核不兼容。
4. **文件处理性能**：大文件（视频/大PDF）采用“前端分片上传+服务端合并”，通过RabbitMQ异步处理，避免阻塞前端操作；MinIO独立部署，提升附件存储/下载速度。

### 四、为什么选择这套技术栈？
1. **纯网页端全覆盖**：一套代码适配平板（一线操作）+ 电脑（领导审阅），无需开发原生APP，降低开发维护成本；
2. **离线功能满足需求**：通过PWA+IndexedDB实现“无网填表格、暂存附件”，完全匹配监狱无网络场景；
3. **兼容银河麒麟**：所有服务端组件均适配银河麒麟服务器，满足内网私有化部署和合规要求；
4. **报告生成精准**：通过Apache POI+Freemarker实现报告自动填充，完美对接《派驻检察室月度工作情况报告》的字段需求；
5. **成熟稳定**：所有技术均为开源主流方案，社区支持强，排查问题方便，后续维护成本低。

### 总结
这套纯网页端技术栈的核心是「**前端适配+离线能力+银河麒麟兼容+报告自动生成**」，既满足“平板无网填表格、联网自动上传”的核心需求，又能通过服务端自动收集数据生成最终月度报告，同时适配银河麒麟服务器的部署要求，是纯网页端方案的最优选择。
